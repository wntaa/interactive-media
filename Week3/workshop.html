Hunt'N'Gather

Web Archives & Galleries

Internet Archive

archive.org/web

The Internet Archive's Wayback Machine is a digital archive of the World Wide Web containing over 735 billion web pages, allowing users to access historical versions of websites by date through a timeline-based navigation system.

Organisation Method

The Wayback Machine organises its vast collection chronologically, using a calendar interface where users can select specific dates to access snapshots of websites.

UbuWeb

ubu.com

UbuWeb is a completely independent resource dedicated to avant-garde art, featuring thousands of works including sound art, visual poetry, film, and critical texts that would otherwise be inaccessible, organised through a deliberately minimalist interface that reflects the experimental nature of its content.

Organisation Method

UbuWeb organises its collection primarily by medium (film & video, sound, contemporary, historical, papers, etc.), then by artist name alphabetically within each section.

MoMA Collection

moma.org/collection

The Museum of Modern Art's digital collection presents over 200,000 artworks through a sophisticated database interface that allows visitors to explore connections between artists, movements, and time periods.

Organisation Method

MoMA's digital collection employs a faceted classification system allowing users to filter works by department, classification, artist, date, and other metadata.

Commercial Site Analysis

TEMU.com

temu.com

TEMU.com employs a product-dense, grid-based e-commerce layout designed to showcase maximum items. The structure follows a vertical hierarchy: fixed navigation header, promotional banners, category navigation, and an infinite-scroll product grid that encourages continued browsing.

Colour Palette
Typography
  • Headline Font: Poppins (sans-serif)
  • Body Font: Inter (sans-serif)
  • Price Font: Roboto (semi-bold)

Interactive Elements

Megamenu Navigation

Expandable category menus appear on hover, providing access to product categories without requiring page loads. Implemented using CSS transitions with JavaScript-powered dropdown expansion.

Carousel Banners

Auto-rotating promotional image sliders with manual navigation dots showcase multiple promotions in limited space. Utilises JavaScript-based timing and transition effects with touch/swipe support.

Infinite Scroll

Continuous loading of products as user scrolls down eliminates pagination and encourages longer browsing sessions. Implemented with Intersection Observer API to detect viewport position and trigger AJAX loading.

Price Countdown Timers

Animated timers showing limited-time offers create urgency for purchasing decisions. Uses JavaScript setInterval updating DOM elements in real-time to display countdown.
