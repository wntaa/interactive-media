Pre-Workshop

Exploring Reflective Interactions

In preparation for this week's workshop, I investigated creative coding projects that engage with reflection, webcams, and mirror-like interactions. These examples demonstrate how digital interfaces can create novel reflective experiences, transforming how we see ourselves through technology.

Hunt'N'Gather

Noteworthy Webcam & Mirror Projects

Pixelated Mirror

editor.p5js.org/jht1493/sketches/3twOKVv-j

This interactive piece utilises a webcam feed to create a pixelated mirror effect. As users move in front of their webcam, their image is displayed in large, colourful pixels, offering a distorted yet recognizable reflection.

Technical Specifications

  • Technology: Built using p5.js, a JavaScript library designed for creative coding
  • Input: Webcam feed
  • Processing: Video pixels are sampled and enlarged to create the pixelation effect

User Interaction

Users see a real-time, pixelated version of themselves, encouraging playful movement and exploration of how their actions affect the visual output. The sketch processes webcam input without requiring additional controls, creating an intuitive mirror-like experience.

Sequins Mirror

editor.p5js.org/themis/sketches/Ga1Nn1yBM

This project utilises a webcam feed to create an array of circles that mimic sequins, reflecting the user's movements in real-time. Each circle represents a pixel from the video feed, and when interacted with, they fall, creating a dynamic visual effect.

Technical Specifications

  • Technology: Built using p5.js, a JavaScript library for creative coding
  • Input: Webcam feed combined with mouse interaction
  • Processing: Video pixels are mapped to circular elements that respond to both video input and mouse events

User Interaction

Users can interact with the sequins by pressing or dragging the mouse over them, causing the circles to fall and creating an engaging visual experience. This dual interaction model combines passive reflection with active manipulation, allowing users to both observe themselves and modify the resulting visual representation.

Webcam Experiment

reddit.com/r/creativecoding

An experimental project that manipulates webcam input to create unique visual effects, transforming the live video feed into an interactive artwork.

Technical Specifications

  • Technology: Utilises p5.js for real-time video manipulation and creative visual output
  • Input: Webcam feed
  • Processing: Algorithmic transformation of video data into abstract visual patterns

User Interaction

Users' movements are captured and transformed, allowing them to see themselves represented in novel and artistic ways, encouraging exploration of motion and form. The system creates an abstract reflection that responds to the user's presence and movement, creating a feedback loop between physical action and digital representation.

Summary

Creative Coding with Reflective Interfaces

These three creative coding projects demonstrate innovative approaches to digital mirrors and reflection:

  • Pixelated Mirror transforms webcam input into a gridded, pixel-art style representation that maintains recognizable forms while abstracting detail.
  • Sequins Mirror converts video data into interactive circular elements that respond both to the user's appearance and direct manipulation through mouse input.
  • Webcam Experiment creates artistic distortions of video feed, offering novel ways to visualize human movement and presence.

All three projects use p5.js as their technical foundation, demonstrating the library's versatility for creative webcam applications. They share a common theme of transforming our self-image through digital processing, creating a dialogue between physical presence and digital representation. These examples show how creative coding can reimagine the concept of mirrors, offering new perspectives on how we see ourselves through technology.
