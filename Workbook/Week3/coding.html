Week 3 Learning Focus

CSS Grids

CSS Grid Layout is a two-dimensional layout system designed for organising content in rows and columns. Unlike Flexbox which is primarily one-dimensional, Grid allows precise control over both horizontal and vertical alignment, making it ideal for creating complex layouts and responsive designs.

Responsive Design

Responsive web design ensures that websites adapt to different screen sizes and devices. This approach uses fluid grids, flexible images, and media queries to create layouts that respond to the user's viewport, providing an optimal viewing experience across desktops, tablets, and mobile phones.

Media Queries

Media queries allow CSS to be applied conditionally based on device characteristics like screen width, height, or orientation. They are essential for responsive design, enabling developers to create different styling rules for various devices and viewport sizes, enhancing the user experience across all platforms.

Github Pages

GitHub Pages is a static site hosting service that takes HTML, CSS, and JavaScript files directly from a repository on GitHub. It allows for easy deployment of websites directly from your GitHub repository, making it simple to share and showcase web projects with a custom domain option.

File Paths

Understanding Link Structure

./ — This single dot beginning essentially says "In this folder" to your browser.

Example: <script src="./script/script.js"></script>

../ — This two-dot beginning tells the browser to move up a folder before beginning its journey to the destination.

Example: <a href="../index.html">Back to home</a>
