3D GENERATIVE ART

Pre-Workshop

Exploring 3D Generative Experiences

In preparation for this week's workshop, I investigated three distinctive 3D generative art projects that showcase different approaches to computational creativity. These examples demonstrate how algorithms, user interaction, and emerging technologies can create immersive, evolving visual experiences.

Noteworthy 3D Generative Art Works

Silk (2016)

weavesilk.com

Silk is an interactive generative art experience where users draw with symmetrical, flowing, luminous threads that create magical 3D-like patterns resembling flowers, stars, and organic structures.

Technical Specifications

  • Technology: JavaScript with HTML5 Canvas and WebGL
  • Creator: Yuri Pederick
  • Release: 2016

User Interaction

Highly interactive; users create artwork by dragging their cursor/finger across the screen, with controls for symmetry types (mirror, rotational), color palettes, opacity, and pattern complexity; the system continuously generates flowing silk-like strands that follow cursor movement with physics-based animations.

Controls

Mouse/touch input for drawing, UI buttons for symmetry options (1-fold to 12-fold), color palettes, and mirror modes.

Neural Synesthesia (2018)

memo.tv/works/neural-synesthesia

Neural Synesthesia combines machine learning with real-time 3D visualization, where a neural network trained on associations between sound and visual patterns creates flowing, organic forms that respond directly to music and audio input.

Technical Specifications

  • Technology: Three.js with TensorFlow.js, WebGL for 3D rendering, Web Audio API
  • Creator: Memo Akten
  • Release: 2018

User Interaction

Semi-interactive; users can provide audio input through microphone or music files, adjust model parameters like complexity and responsiveness, and choose different pre-trained neural network models that interpret audio differently.

Controls

Audio input controls (microphone/file upload), parameter sliders for neural network behavior, camera controls for viewing the 3D forms from different angles.

Electric Sheep (1999-present)

electricsheep.org

Electric Sheep is a collaborative, evolving artwork where users' computers run a distributed fractal flame algorithm to generate "sheep" (abstract animated fractal forms), which then "breed" and evolve through a user-driven selection process.

Technical Specifications

  • Technology: Custom C++ rendering engine with distributed computing framework, fractal flame algorithms
  • Creator: Scott Draves
  • Release: 1999 (continuously evolving)

User Interaction

Collaborative interaction; users participate in a distributed computing network, contributing processing power to render frames of the animations. They can vote on which "sheep" they prefer, influencing the evolutionary algorithm that determines which patterns survive and "breed" to create new variations.

Controls

Voting interface (up/down votes for preferred animations), optional parameter adjustment for users who want to create and submit their own genetic code for new sheep, screensaver mode for passive viewing.

Summary

Approaches to 3D Generative Art

These three generative art projects demonstrate different approaches to computational creativity and user engagement:

  • Silk emphasizes direct manipulation and immediate feedback, allowing users to actively create flowing patterns through intuitive drawing interactions.
  • Neural Synesthesia explores the intersection of AI and art, creating a symbiotic relationship between machine learning models and human audio input.
  • Electric Sheep pioneered distributed collaborative creation, where thousands of computers work together to evolve complex fractal animations guided by collective human aesthetic preferences.

These works represent a spectrum of user agency in generative art: from direct creation (Silk), to parameter adjustment of AI systems (Neural Synesthesia), to participation in collective evolution (Electric Sheep). They also showcase different technical approaches, from real-time browser-based rendering to distributed computing networks, demonstrating the diverse pathways for creating compelling 3D generative art experiences.
