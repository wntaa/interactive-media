Week 6

WORKSHOP ACTIVITY

Object Categorisation

Exploring Layout & Hierarchy

In this workshop activity, we explored principles of layout and hierarchical organisation by arranging items from a bag based on specific categorsation criteria. This exercise demonstrated how physical arrangement can communicate information and how organisational systems can reflect both logical and playful approaches to classification.

Categorical Organisation

We first organised the items by conventional categories, grouping similar objects together. This created a logical hierarchy based on object type, material, or function. This systematic approach is commonly used in information design to create clear, understandable visual structures.

Playful Interpretation: "Cat Appeal" Layout

For the second arrangement, we explored a more imaginative categorisation: how likely cats would be to play with each object. This unconventional organising principle led to creative layout decisions, including:

  • Using radial arrangement with high-interest objects placed farther from center
  • Creating deliberate disorder for highly appealing items (mimicking post-play chaos)
  • Maintaining neat arrangements for low-interest items

This approach demonstrates how layouts can communicate not just categories but also narrative, behavior, and hypothetical scenarios.

Reflection

Layout as Communication

This workshop activity demonstrated several key principles about layout and organization:

  • Physical arrangement as information: How we position objects can communicate relationships, hierarchies, and categories without requiring explicit labels.
  • Multiple organizational systems: The same set of objects can be meaningfully arranged in different ways depending on the criteria being emphasized.
  • Order vs. chaos as communication: The deliberate use of disorder (for items cats would play with) shows how the degree of organization itself can communicate meaning.
  • Narrative through arrangement: The cat-focused layout tells a story about feline behavior through spatial organization alone.

These principles connect directly to digital interface design, where arrangement of elements communicates importance, relationships, and user pathways. The playful "cat appeal" categorization also demonstrates how unconventional organizing principles can generate creative and engaging visual solutions.

Application to Digital Design

From Physical to Digital Organization

The principles explored in this physical arrangement exercise translate directly to digital interface and information design:

Digital Applications

  • Content hierarchy: Just as we arranged physical objects by importance, digital interfaces use size, position, and visual weight to establish hierarchy.
  • Information architecture: The categorical organisation of objects mirrors how websites organise content into sections and subsections.
  • User-centered organisation: The "cat appeal" arrangement demonstrates perspective-based organszation, similar to how interfaces might reorganize based on user behavior or preferences.
  • Visual storytelling: The progressive disorder in the cat-focused layout shows how visual arrangement can communicate narrative in both physical and digital spaces.

This workshop activity reinforces that strong organisational principles underpin both physical and digital design, with spatial relationships serving as a fundamental communication tool across mediums.
