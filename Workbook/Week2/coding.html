Key Learnings

CSS Layout

Understanding how to create multi-column layouts using CSS Flexbox, which allows for responsive design and flexible content arrangement.

CSS Selectors

Working with different CSS selector types including element selectors, class selectors (with dot notation), and ID selectors (with hash notation).

Box Model

Implementing margin, padding, and width properties to control spacing and dimensions of elements, creating visual separation between content areas.

CSS Units

Using various CSS units like rem for typography, percentage for responsive dimensions, and viewport units (vw/vh) for full-screen layouts.

Custom Styling

Applying background colors, font family specifications, and text alignment to create visually distinct sections with proper typography.

HTML Structure

Building semantic HTML structure with header elements and proper content organization using divs with appropriate classes and IDs.
