Key Learning Concepts

During the Week 1 workshop, I gained fundamental knowledge of HTML structure and syntax. Here are the main concepts I learnt:

HTML Document Structure

I learnt that every HTML document follows a standard structure with DOCTYPE declaration, html, head, and body elements. The head contains meta information whilst the body contains the visible content.

Text Formatting Tags

I practised using various text formatting tags like <strong>, <em>, <i>, and <q> to style text content in different ways. This helps create visual hierarchy and emphasis in web content.

Basic Interactivity

I implemented a simple alert function with JavaScript and a button element to understand how HTML can interact with scripting languages to create interactive experiences.

My Development Process

This timeline shows my progression from starting with a blank file to creating a functional HTML page during the workshop activity:

Step 1: Basic Structure

I started by setting up the basic HTML structure with DOCTYPE, html, head, and body tags as demonstrated in class. This forms the skeleton of any web page.

Step 2: Adding Content

Added headings, paragraphs, and applied text formatting with strong and em tags. I chose to include content about what we were learning in class to make it relevant.

Step 3: Adding Images

Incorporated an image element with a URL source. I had to research the correct syntax for the img tag and understand that it's a self-closing tag.

Step 4: Adding Interactivity

Created a button element and linked it to a JavaScript function to display an alert message. This was my first experience connecting HTML with JavaScript.

Step 5: Layout Structure

Created a two-column layout using flexbox for the Evil Media excerpt. This helped me understand how to arrange content spatially on a webpage.

Challenges & Solutions

During the development process, I encountered several challenges. Here's how I resolved them:

Image Display Issues

The image wasn't displaying correctly at first. I could see a broken image icon in my browser.

Solution:

I realised I had forgotten to close the img tag properly. I learnt that img tags are self-closing and should end with /> instead of having a separate closing tag.

Text Layout Challenges

When trying to create the two-column layout for the Evil Media excerpt, the text was appearing stacked vertically instead of side by side.

Solution:

After researching, I discovered I needed to use CSS flexbox styling to create columns. I added inline CSS with display: flex and flex-direction: row to arrange the divs horizontally.

Alert Button Not Working

Initially, clicking the button didn't trigger any alert.

Solution:

I had placed the script tag in the wrong location and misspelt "function" in my JavaScript. After correcting these issues and making sure my function matched the onclick attribute, the alert worked properly.

Personal Reflection

Week 1 marked my first formal introduction to web development. As someone with limited prior coding experience, I found the process both challenging and exciting.

The most valuable insight for me was understanding that HTML is essentially a semantic language - it's about giving meaning to content through structure. The idea that we're not just styling text, but defining what each piece of content represents (headings, paragraphs, emphasis, etc.) helped me approach coding with a clearer mindset.

I was particularly intrigued by how quickly we could create a functional webpage with just a few tags. The immediate visual feedback when previewing the HTML file made the learning process more engaging compared to other programming languages where results might not be as immediately visible.

Moving forward, I want to explore how to create more complex layouts and incorporate design principles into my HTML work. I'm also curious about how HTML interacts with CSS for styling and JavaScript for functionality - elements we've only touched on briefly so far.
