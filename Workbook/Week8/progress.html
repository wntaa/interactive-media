Trash Website

This experimental website concept allows users to interact with web pages using gestures instead of mouse clicks. Inspired by JODI.org's chaotic layout, the user navigates through multiple sites in an intentionally absurd or fragmented way.

The twist: instead of clicking links, the site listens to the user’s gestures — specifically swipe motions detected via webcam. This creates a strange, playful experience where users are unsure if or how their input is being understood.

Interaction logic: webcam detects hand x-position. A swipe left or right reloads the page to a new destination.

let lastX = null;
function checkSwipe(currentX) {
  if (lastX !== null) {
    const dx = currentX - lastX;
    if (dx > 0.05) {
      window.location.href = "page2.html";
    } else if (dx < -0.05) {
      window.location.href = "page1.html";
    }
  }
  lastX = currentX;
}

Potential challenge: implementing gesture detection accurately via webcam may require ML libraries or third-party APIs. A fallback option is using keyboard input for interaction testing.

Photo Booth

This web-based photo booth uses the webcam to let users take a picture and decorate it. Tools include a pencil for drawing, drag-and-drop stickers, and other playful interface elements. The design focuses on personalising captured moments directly within the browser.

Abstract Drawing with Music | SOUNDPAINTER

This concept turns a user into a visual composer. When pressing keyboard keys (e.g. 1, 2, 3), each plays a unique audio clip (e.g. piano, bass, percussion).

Simultaneously, shapes or lines are drawn onto the canvas, reacting to pitch and volume: higher pitch = larger size, louder volume = more vibrant colour. This creates a musical drawing performance.
