Topics Covered

Week 5 Learning Focus

Image & Video Processing

P5.js offers robust tools for loading, displaying, and manipulating images and video. Learn how to apply filters, access pixel data, and create dynamic visual effects by processing visual media in real-time.

Sound Integration

With the p5.sound library, you can load, play, and analyse audio in your sketches. Create interactive sound experiences by connecting visual elements to audio properties like amplitude and frequency, enabling audio-responsive animations.

Webcam & Camera

Capture live video from the user's webcam and incorporate it into your sketches. This opens possibilities for interactive installations, augmented reality experiences, and computer vision applications when combined with libraries like ml5.js.

Advanced Interactions

Move beyond basic mouse interactions with keyboard events, gesture recognition, and state management techniques. Create more complex user experiences by combining multiple interaction methods and controlling the flow of your application.

Step-by-Step Guides

Adding Images

1. Drag and drop your image onto the editing pane p5 Processing sketch. This will make a copy of the image in a folder titled 'data', which lives in the same folder as your sketch file {CMD K}.

2. Declare your image variable, and point to the image file in your sketch, i.e.

var img1; function preload() { img1 = loadImage("data/yourimage.jpg"); }

3. You can then use your image across any function, eg you can display your image in your sketch by adding:

image(img1, 0, 0);

inside your draw() {} function

Add Webcam

At the beginning declare your capture variable:

var capture;

And in your setup():

capture = createCapture(VIDEO); capture.size(640, 480);

Then place in your draw(), with or without the filter() function:

image(capture, width / 2 - 320, height / 2 - 240); filter(THRESHOLD);

Dot To Dot

Press any key on the keyboard to place a number, click and drag your mouse to draw.

var counter; function setup() { createCanvas(windowWidth, windowHeight); background(255,255,255); counter=0; textSize(20); } function draw() { if (mouseIsPressed) { strokeWeight(10) stroke(255,0,0); line(mouseX, mouseY, pmouseX, pmouseY); } } function keyPressed(){ counter++; fill(0); circle(mouseX, mouseY, 5); text(counter, mouseX,mouseY); }

POINTILLISM

let img; let pixelSize = 10; function preload() { img = loadImage('data/jaggedOrbit.png'); } function setup() { createCanvas(windowWidth, windowHeight); background(0); noLoop(); } function draw() { image(img, mouseX-img.width/2, mouseY-img.height/2, img.width, width/img.width*img.height); for (let x = 0; x < img.width; x += pixelSize) { for (let y = 0; y < img.height; y += pixelSize) { let c = img.get(x, y); fill(c); noStroke(); let px = random(x - pixelSize / 2, x + pixelSize / 2); let py = random(y - pixelSize / 2, y + pixelSize / 2); ellipse(px, py, pixelSize, pixelSize); } } }
